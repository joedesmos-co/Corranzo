# Dense OMR note-matching iteration

## Root cause

MIDI-only deduplication in vector note grouping (`dedupeNotesByMidi`) collapsed spatially distinct noteheads that mapped to the same pitch. On Cruel Angel this removed **~105 notes** between detection (2791) and emission (2686).

## Fix

`src/features/omr/omrNoteDedupe.js` ‚Äî spatial dedupe key: `clef:midi:cxBucket:cyBucket` (6px buckets). Applied in `groupVectorNoteheads`, `mergeGroupsSharingBeat`, and chord coalesce paths in `processVectorOmrPage.js`.
